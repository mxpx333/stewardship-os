My Recommendation: Build These Next

1 Weekly Review Page

A separate view (or modal) that shows: time spent per category this week, ships progress, flow completion rates, deep work total, and a guided "what went well / what to improve" reflection. Data is already in Google Sheets — we just need to visualize it.

2 Time Tracking by Category

Add a "category" field to deep work blocks (e.g., "Pipeline", "Outreach", "Internal", "Strategy"). Auto-calculate weekly time per category. Show a simple pie or bar chart on the weekly review.

3 Streak Visualization

For each daily routine item, show current streak and 30-day completion rate. Add a small calendar heat map (like GitHub contribution graph) to the stats. Motivates consistency on prospecting and quiet time.

4 Buffer Time Indicators

After each calendar meeting, show a subtle "buffer" zone in the timeline (5-10 min). Not blocking — just a visual reminder to do follow-ups before jumping to next thing. Add a "Post-meeting notes" quick capture.

5 Dark/Light Mode Toggle

Store both color palettes, add a toggle in the top bar. Low effort, high user satisfaction. We already have both themes built — just need a switch.

6 Keyboard Command Bar

Press Cmd+K to open a quick-entry bar. Type "add task: Review proposal 60m" or "start deep work" or "log clearing: printer setup". Rapid entry without clicking through modals.
